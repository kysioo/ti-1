## Babel6

* http://babeljs.io/env[babel-preset-es2015 -> babel-preset-env]

[source,sh]
----
npm install babel-cli --save-dev

mkdir babel6-project
cd babel6-project

npm init -y
----
Add _.babelrc_ file:
[source,json]
----
{
  "presets": [
    ["env", {
      "targets": {
        "chrome": "66"
      }
    }]
  ],
  "sourceMaps": "inline"
}
----
Run:
[source,sh]
----
npm install babel-preset-env --save-dev
----
